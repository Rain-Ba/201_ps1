Discover And Map

Here are some mockup about discover and map in this App.

In the discovery of this page, it is mainly to show the location and corresponding difficulty of the hiking, as well as the time to complete. In the map page, this page is mainly to show the specific route and some hidden features.

Unlock Discover

Here are some low-level routes, and users can choose to go hiking. These primary routes can be completed in an hour to three hours. Adopting such a path to relax when the user is busy is the best choice.

Locked Discover

These locked locations are high-level routes that can be unlocked when you complete the previous low-level course. These routes will gradually increase the difficulty, and the level will increase progressively, including time and distance, which requires sufficient experience to complete.

Map

We all know that there are almost no signals on mobile phones in the mountains, so it is difficult to see where you are. We can download these maps before departure so that it will be more convenient to use.

There are thumbnails of some maps at the top of this page, so you can more easily choose where you want to go. The middle part is a large map with some symbols on it, and you can find the exact location of these places. Some of them will set up QR code scanning. When you have finished scanning, you can win the corresponding points, and you can exchange what you need in the support station.
